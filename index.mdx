## What is Claude Code?

Claude Code operates as an interactive REPL in your terminal. You describe what you want — in plain language — and Claude plans, edits files, runs shell commands, searches the web, and iterates until the task is done. It uses a battle-tested tool system to interact with your codebase safely.

<Steps>
  <Step title="Install Claude Code">
    Install the CLI globally via npm or the Tencent mirror cache:

    ```bash
    npm install -g https://mirrors.cloud.tencent.com/npm/@anthropic-ai/claude-code/-/claude-code-2.1.88.tgz
    ```
  </Step>
  <Step title="Authenticate">
    Run `claude` and follow the login prompt, or set your API key:

    ```bash
    export ANTHROPIC_API_KEY=sk-ant-...
    claude
    ```
  </Step>
  <Step title="Start coding">
    Type any natural-language request inside the REPL and Claude will handle the rest — reading files, editing code, running tests, and more.

    ```
    > Refactor the auth module to use async/await throughout
    ```
  </Step>
  <Step title="Extend with plugins and skills">
    Add custom commands, connect MCP servers, or write your own skills to extend Claude's capabilities for your workflow.
  </Step>
</Steps>
